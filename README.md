# Transcreve Facil Privado v14

Versao estavel de recuperacao, com codigo-fonte ASCII/UTF-8 limpo para evitar erro de codificacao no Streamlit Cloud.

## Arquivos obrigatorios

- app.py
- requirements.txt
- packages.txt
- runtime.txt
- README.md
- assets/logo_full.png
- assets/logo_icon.png

## Atualizacao recomendada

1. No GitHub, apague o app.py antigo.
2. Apague a pasta __pycache__, se existir.
3. Apague arquivos soltos estranhos, como download.
4. Envie os arquivos desta versao.
5. Commit changes.
6. No Streamlit Cloud: Manage app > Reboot app > Clear cache.
